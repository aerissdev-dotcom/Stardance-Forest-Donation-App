# Stardance-Forest-App-Repo
A simple website UI (without backend) for donations towards forests and their growth, a project focused on Stardance Challenge.

# Features
Minimalist, dual-panel layout with forest imagery
Donation form with real-time validation (credit card, CVC, expiration date)
Live simulated donation counter with random contributions
Fully responsive design for mobile, tablet, and desktop
Built with pure HTML, CSS, and vanilla JavaScript – no frameworks

# How to Run Locally
Clone this repository

Open index.html in your browser

No dependencies, no build steps – just open and go

- Vanilla JavaScript (ES6+)
