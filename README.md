# Personal Portfolio

A clean, minimal personal portfolio website built with pure HTML and CSS. No frameworks, just fast, responsive pages.

## Pages

- **Home** – Introduction, about me, quick stats
- **Projects** – KeyGlow and ReFlow with descriptions and links
- **Contact** – Email, GitHub, social links

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Inter font via Google Fonts

## Live Demo

https://aerissdev-dotcom.github.io/Stardance-Portfolio-App/index.html

## How to Run Locally

1. Clone the repo
2. Open `index.html` in your browser
3. That's it – no build steps, no dependencies
